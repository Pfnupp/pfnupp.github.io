kurze Infos

Installation:
  - TestPlugin.jar in Server\Plugins kopieren.
  - PfnuppPack.zip in %appdata%\.minecraft\resourcepacks kopieren
    (MCPatcher wird benötigt!)

Befehle (als OP):
  /dreckfuss  Aktiviert/deaktiviert das Setzen von Erde unter dem Spieler (beim Laufen).
              Der Status wird zum Spieler gespeichert und ist nach einem Relogin unverändert.

Zusätzlicher Block:
  Sand (12) mit Metadata 10 (und eigenem Bild in ResourcePack)
  Wird der Block beim Laufen in der Hand gehalten wird ein Dia-Block beim Laufen unter dem Spieler gesetzt.
  (War zum Test für den Zugriff aufs Inventory eines Spielers und das Auslesen der SubID eines Blocks)
